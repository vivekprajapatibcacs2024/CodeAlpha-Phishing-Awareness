# Internship Task 1 – CodeAlpha-Phishing-Awareness 
Phishing Awareness Training Module – Cyber Security Internship Project

This project was completed as part of the Cyber Security Internship at CodeAlpha.

Project Overview

The Phishing Awareness Training Module is designed to educate users about phishing attacks and how to prevent them.
The module covers:
Introduction to phishing
Types of phishing attacks
Social engineering techniques
How to identify phishing emails
Prevention and protection methods

Objective

The main objective of this project is to spread awareness about phishing attacks and promote safe cybersecurity practices.
